# Crypto Product Intelligence & Quality Analytics Platform (2026)

Built an end-to-end fintech analytics platform processing **200+ crypto assets** across 1H, 24H, and 7D timeframes for data-driven investment decisions.

## Key Features
- Product-grade KPIs covering momentum, average downfall risk, liquidity, volume, and market capitalization.
- Side-by-side asset comparison with strict input validation.
- Liquidity concentration analysis (Top-5 vs Others).
- Executive-ready dashboards in Power BI and Excel using dynamic slicers and actionable summaries.
- Enterprise-style quality and governance practices including data validation, regression checks, access controls (VBA), Agile tracking (JIRA/Confluence), and version control (GitHub).

## Tools
Python, SQL, Power BI, Excel (VBA), VS Code, JIRA
